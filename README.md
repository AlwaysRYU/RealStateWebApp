# 👊 죽이는 방찾기, 죽방
삼성청년소프트웨어아카데미 6기 1학기 파이널 프로젝트

## 🛠 프로젝트 개요
- 로그인 / 게시판 글 작성기능이 있는 커뮤니티 기능이 있는 사이트
- 데이터를 불러와 카카오맵API를 사용해서 사용자에게 직관적으로 매물을 보여줄 수 있는 사이트

## 👨‍👨‍👦 팀원

- 최호준
